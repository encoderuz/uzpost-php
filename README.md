# Uzpost PHP ClientUzpost PHP Client is a simple package to interact with the Uzpost API using GuzzleHTTP. This package allows sending API requests, handling authentication, and logging API interactions.## InstallationYou can install the package via Composer:```bashcomposer require encoderuz/uzpost-php```## UsageInitialize the Client:```phpuse Encoderuz\Uzpost\Http\UzpostClient;$client = new UzpostClient('your-api-key');```
